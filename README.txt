
# AI-Powered Manhwa Scraper
## How to Use
1. Install Python 3.x from [python.org](https://www.python.org/downloads/).
2. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Run the scraper script:
   ```
   python scraper.py
   ```
4. Translated titles will be saved in a file called `translated_titles.txt`.
5. Update the URL and CSS selectors in `scraper.py` to match your target website.
